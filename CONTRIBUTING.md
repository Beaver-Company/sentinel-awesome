# Contribution Guidelines

Here are some guidelines about pull requests for Awesome Sentinel:

- Do not add new files, just modify the `README.md` file.
- When adding an entry, you can provide a short description for your component, as well as a link to your repository. An example:

    - [Sentinel DataSource Nacos](https://github.com/alibaba/Sentinel/tree/master/sentinel-extension/sentinel-datasource-nacos): A Sentinel data source extension based on Nacos (by @sczyh30)

- Keep descriptions short and simple.
- Make sure your text editor is set to remove trailing whitespace and blank lines.
